# Customer-Behavior-Data-Analyst-Portfolio-Project
End-to-end Data Analytics project using Python, SQL (PostgreSQL/MySQL/SQL Server), and Power BI with EDA, data cleaning, business insights, and interactive dashboard reporting.

# 📊 Data Analytics End-to-End Project

## 📌 Overview

This project demonstrates an **end-to-end Data Analytics workflow**, starting from data loading in Python to generating business insights through SQL queries and an interactive Power BI dashboard.

The goal of this project is to showcase practical skills in:

* Data loading
* Exploratory Data Analysis (EDA)
* Data cleaning
* SQL querying across multiple databases
* Data visualization
* Reporting and presentation

This project reflects a **real-world analytics pipeline** used by data analysts in industry environments.

---

## 📂 Dataset

The dataset used in this project contains customer purchase and transaction-related information such as:

* Customer details
* Product category
* Purchase amount
* Payment method
* Location
* Review ratings
* Purchase frequency
* Seasonal behavior

The dataset was analyzed to identify **customer behavior patterns, product performance, and purchasing trends**.

## 🔄 Project Workflow / Steps

### 1️⃣ Data Loading (Python)

* Imported dataset using **Pandas**
* Checked structure and column types
* Handled missing values
* Verified dataset integrity

### 2️⃣ Exploratory Data Analysis (EDA)

Performed:

* Summary statistics
* Distribution analysis
* Correlation checks
* Trend identification
* Category-level comparisons

Used:

* Matplotlib
* Seaborn

---

### 3️⃣ Data Cleaning

Handled:

* Missing values
* Duplicate records
* Incorrect formats
* Column standardization
* Feature creation (derived columns)

Prepared dataset for **accurate SQL querying and dashboard creation**

---

### 4️⃣ SQL Analysis (PostgreSQL / MySQL / SQL Server)

Executed analytical queries such as:

* Top selling products
* Average product ratings
* Customer purchase trends
* Seasonal sales insights
* Payment method usage analysis
* Location-wise purchase behavior

This step helped simulate **real-world database analytics scenarios**

---

### 5️⃣ Power BI Dashboard Creation 📊

Built an **interactive dashboard** including:

* Sales trends
* Customer demographics insights
* Category performance
* Purchase frequency patterns
* Location-based analytics
* Payment method distribution

Dashboard enables stakeholders to quickly understand key insights visually.
